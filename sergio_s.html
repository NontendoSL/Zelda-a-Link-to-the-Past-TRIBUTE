Documents

One of the task of game designer it’s document everything so that as many programmers as artists can go at any time. These documents have to be concise brief and clear, because if you don’t explain well in them could be confusions and generate errors and have to go changing the design or explain the same thing every time.

Like that of the photo with all the documents of the game, all these documents end up converging in one… The Game Design Document(GDD).
